3DS Homebrew Guide

This section will focus primarily on setting up Boot9strap. Keep the device turned off and instert the system SD card into your computer.

Copy SafeB9SInstaller.firm to the root of the SD card and rename it to boot.firm. Then copy boot.3dsx to the root. After both files are copied make a folder called boot9strap on the SD card. Copy boot9strap.firm and boot9strap.firm.sha into that folder on the SD card.

Once that is done, eject the SD card and insert it into the system. You can turn the sytem on to find the spot that puts the system in sleep mode now if you need to find it again. Once that is done turn the system off and start it holding Start, Select, X, Power for a few seconds. This might take a few tries to get right but when you do your system should show the SafeB9Installer menu.

Once all of the safety checks are done, you can take the device out of sleep mode, and enter the code to install boot9strap. After it installs, hold the power button to turn the system off. Remove the system SD card and insert it into your computer. Delete the boot.firm on the root of the SD card and replace it with the boot.firm from the Luma3DS files you downloaded earlier.

We are almost done, all the firmware we need is installed, so we just need to get some utilities installed to make the system useful:

  • Anemone3DS (cia file)
  • Checkpoint (cia file)
  • Homebrew Launcher Wrapper
  • GodMode9
  • DSP1
  • FBI (both cia and 3dsx files)
  • ctr-no-timeoffset
  • Luma3DS Updater (cia file)

On the system SD card, make two folders on the root. Call one 3ds, and the other one cias. Copy all of the cia files you downloaded into cias, and all of the 3sdx files into 3ds. Make a third folder called payloads in the luma folder on your SD card, then copy GodMode9.firm into it. Then, copy the gm9 folder from the GodMode9 folder onto the root of the SD card.

Insert the SD card into your system and restart the device. Open Download and Play on your system and press L, Down, and Select at the same time, this should open the Rosalina menu. Select “Miscellaneous options”, then "Switch the hb. title to the current app". Press B three times to exit the Rosalina menu. Restart Download and Play, it should launch the homebrew menu for now until we can make an icon on the home screen for it.

Scroll down and find ctr-no-timeoffset and launch it, then press A and Start. This makes some adjustments to the system clock the we will finalize in a bit. Once back in the Homebrew Launcher, find FBI in the menu and launch it. Select SD, then cias, then select "Install and delete all CIAs". You can now press home and close Download and Play.

You should have some new applications on your home screen, the first one you need to launch is DSP1. Let it run and press B when it is done to exit.
