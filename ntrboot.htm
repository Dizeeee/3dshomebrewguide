3DS Homebrew Guide

As stated earlier, you will need a flash cart in order to perform the ntrboot method. If you don't already have a flash cart, there is a useful table on 3DS Hacks Guide that shows the pros and cons of various flash carts. Make sure that the one you pick is currently supported. If you do want to purchase one, it is highly reccomended to use nds-card as it is basically the only reputable seller. You can find certain flash carts listed on Amazon, but you might run the risk of purchasing a counterfeit card that may die while attampting this method.

Make sure thatyou know the type of flash card you have before starting this section, it isnt that hard to tell but several manufacturers use similar branding. Additionally, if you want to restore your flash card to its orignal state after following this guide, please refer to the manufacturer's website if you want instructions on how to reset your flash cart.

Before beginning this section you need to perform a few tests, first of all make sure your flash cart works properly. Simply make sure it launches to the menu properly (You can also try launching an .nds file if you want to). Next, if you are not using an original 2DS, find a magnet and use it to test putting your system into sleep mode. The 2DS has a sleep switch and you have to use that instead. For the other models locate the area that triggers sleep mode by moving the magnet around the A B X Y buttons until the screens go black. Make sure that you can keep it in sleep mode for long enough to restart the system as that is all we need to trigger the payload once it is loaded on the SD card.

Next, we will begin the process of setting up Ntrboot. You will need to install a couple things first:

  • Boot9strap (the ntr version, not the devkit or ntr-devkit versions)
  • Ntrboot Flasher
  • SafeB9Installer
  • Luma3DS
  • The Homebrew Launcher

Once these are downloaded, unzip the folders and put them in your 3DS folder, preferrably in folders labling what they are. Make sure your device is off and open the flash cart SD card on the computer. Then, make a folder called ntrboot at the root of the SD card. Copy the boot9strap_ntr.firm file to the ntrboot folder. Copy the ntrboot_flasher_nds.nds file over to the root of the SD card.

Once that is all done, eject the SD card and put it back in the flash cart. Put the flash cart in your system, launch it, and open Ntrboot flasher. This will start the flashing process, which will overwrite the data on your SD card. If you would like to make a copy you can still close the flasher now and copy the data on the SD card to another backup folder in your 3DS folder.

If you are ready to begin installing Ntrboot, press A to continue. Select the type of flash cart you have using the arrow buttons and the A button. Select "Dump flash" to make an additional backup of your flash cart on the SD card. Select your flash cart again and select "Inject FIRM" to start installing Boot9strap. Then turn the device off and continue to the next section to continue the installation.
